# CPU Battle Royale Simulator

A 2D CPU-driven battle royale simulation game built with Phaser.  
Players (CPUs) move around, fight each other, and a winner emerges.  

## Features
- Solos simulation
- Random CPU movement
- Top-down map view
- Stats tracking (future update)

## How to Run
1. Clone this repository
2. Open `index.html` in a web browser
